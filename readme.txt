Chương trình kiểm tra trực tuyến, dùng cho mạng cục bộ (phòng máy tính):


* Mục đích sử dụng: dùng cho HS
- Kiểm tra trắc nghiệm trên máy
- Làm bài thực hành (nộp lại sản phẩm)

* Cách dùng:
1. Kiểm tra trắc nghiệm: 
- Quản lí bài kiểm tra: mở file baitn.php
- HS làm bài kiểm tra : mở file tracnghiem.php
- Xem kết quả : mở file : ketqua.php

2. Làm bài thực hành:
- Soạn yêu cầu: make.php
- Làm bài & nộp bài: baitap.php

* Lưu ý:
- Chương trình khuyến cáo dùng trong mạng LAN, vì tính bảo mật không cao (đề và đáp án).
- Kết quả được cập nhật liên lục mỗi khi có HS nộp bài
